INSTALL:
Clone Project ini setelah selesai pada terminal masuk kedalam directory project

Ketikkan:

1. Composer Install
2. cp .env.example .env
3. php artisan key:generate
4. php artisan migrate (Jangan lupa set up database)
5. php artisan db:seed
